# Home Cell Weather

Control-center style weather dashboard for Williamsburg, VA (23188).

## Features
- Live KAKQ radar (NWS)
- Current conditions via NOAA API
- 3-day forecast
- Active alerts
- Auto-refresh every 5 minutes

## Deployment
Enable GitHub Pages:
Settings → Pages → Deploy from branch → main → root

Then access:
https://richardcookester-cmd.github.io/vhammer-weather/

## Embed in Google Sites
Use:
Insert → Embed → URL
